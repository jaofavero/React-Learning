# React Project

This is a simple React project.

## Getting Started

1. Clone the repository.
2. Install dependencies:

    ```bash
    npm install
    ```

3. Start the development server:

    ```bash
    npm run dev
    ```
    
# Getting Started Testing

    ```bash
    npm install -D vitest
    ```

## Features

- Modern React setup
- Easy to customize
